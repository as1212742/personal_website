# Personal Website

## Project Documentation
This repository contains the code for my personal website. The website was built using HTML, CSS, and JavaScript and showcases my portfolio, blog, and contact information.

## Setup Instructions
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/as1212742/personal_website.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd personal_website
   ```

3. **Open the `index.html` file in your web browser**:
   ```bash
   open index.html  # macOS
   start index.html # Windows
   xdg-open index.html  # Linux
   ```

4. Enjoy browsing my website!
